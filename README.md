# HqngOrder

HqngOrder is a complete reimplementation of the SBOrder plugin originally developed by BM Studio. The project was decompiled from the original source and rewritten entirely in Kotlin. The rewrite focuses on delivering better functionality, improved performance, and a more maintainable codebase.

The plugin uses TriumphGUI API for user interface components and FoliaLib for multithreading optimization. All major features from SBOrder remain intact. Enhancements have been made to configuration flexibility, GUI customization, and overall system responsiveness.

Key improvements include:
- Expanded configuration options for greater control over plugin behavior
- More customizable GUI elements to suit server-specific needs
- Optimized multithreading for smoother operation under load
- Cleaner code structure for easier future updates and debugging
